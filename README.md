# Neovim Config
Based on [ddrscott/congif-nvim](https://github.com/ddrscott/config-nvim).

## Setup
```
# Backup old config if needed.
mv ~/.config/nvim{,.bak}

# Clone this repo.
git clone git@github.com:aronnebrivio/config-nvim.git ~/.config/nvim

# Install config
cd ~/.config/nvim
./install
```
